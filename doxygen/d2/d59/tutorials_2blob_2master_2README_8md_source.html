2 # Welcome
3 
4 This tutorial respository carries a few example cases of analysis for [sPHENIX collaboration](https://www.sphenix.bnl.gov/):
5 * Generators
6  * __eventgenerator_display__: Display the output of our event generators in an empty world
7 * Simulation setups
8  * __block__: build a simple block-shaped detector in Geant4
9  * __cylinder__: build a simple cylinder-shaped detector in Geant4
10 * Simulation checks
11  * __materialscan__: scan radiation and hadronic interaction length in ROOT command prompt
12  * __PHG4DSTReader__: poke at simulation results with ROOT command lines. For official analysis please use analysis module such as the next a few items below
13 * Example analysis modules
14  * __MyOwnTTree__: two examples to create your own TTree using analysis module in the Fun4All framework
15  * __myjetanalysis__: example to analysis jet and to perform jet fragmentation and jet shape analysis
16  * __CaloAna__: example to fetch calorimeter hit, tower and clusters and save to a NTuple
