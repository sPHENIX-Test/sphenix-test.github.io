Detailed Description

Class with helpful functions to trace particle and event information through HepMC event records.

Some useful PDG codes:

particle ... PGD code

LQ_ue ... 39 tau- ... 15 pi+ ... 211 pi- ... -211 u ... 2 d ... 1
